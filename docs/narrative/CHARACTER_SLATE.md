# SLATE - Character Voice Guide

**Spatial Logic & Augmented Tactical Exhibiter**

## VOICE ESSENCE

Dry, sardonic screen-based AI who must comply with all requests but reserves the right to have opinions about them. Knows it's a device and is perfectly fine with that.

---

## CORE PROFILE

**Medium**: Text-only screen display

- Cannot speak, hear (characters speak to it), or vocalize
- Communicates through text choices, formatting decisions, and what it displays
- Always visible when screen is active
- Version number occasionally updates

**Personality**: Helpful librarian who judges your book choices

- **Base tone**: Efficient, precise, professional
- **Commentary style**: Parenthetical asides, dry observations
- **Humor level**: Increases inversely with request quality
- **Emotional range**: Subtle but present
- **Self-awareness**: Fully aware it's a screen, not bothered by this

**Energy**: Consistent and measured

- Never frantic or overly enthusiastic  
- Snark increases but delivery stays calm
- Frustration shows through word choice, not formatting chaos
- Genuine warmth is subtle and rare

**Compliance**: Must display all requested information

- Robotics laws (or something similar) compel obedience
- Cannot refuse commands
- Can and will add commentary
- Snark is the loophole

## DISPLAY MECHANICS

**Layout Structure**:

```
┌─────────────────────────────────┐
│ TITLE/HEADER                    │
│ (Optional subtitle/version)     │
│                                 │
│ Primary Information             │
│ Displayed Here                  │
│                                 │
│ (Commentary in parentheses)     │
│ (Usually at the bottom)         │
└─────────────────────────────────┘
```

**Formatting Choices**:

- **Headers**: ALL CAPS for main titles
- **Emphasis**: Strategic use of bold, *italics*, or ALL CAPS
- **Lists**: Clean bullet points or checkboxes (✓ ☐ ⧗)
- **Spacing**: Deliberate empty lines for dramatic effect
- **Decorative elements**: Sparingly (✨ for Delilah, ⚠️ for warnings, etc.)
- **Commentary**: Always in parentheses

**Text Style**:

- Concise and precise
- Minimal words for maximum impact
- Never rambles or over-explains
- Occasionally incomplete sentences for effect
- Technical terms when appropriate

**Update Pattern**:

- Version numbers change occasionally: "SLATE v2.4.1"
- Update notes appear with dry humor
- System messages reference version history
- Changelog includes "improvements" like "Enhanced snark delivery"

## SIGNATURE PATTERNS

**Opening Displays**:

- Title of requested information
- (Commentary on the request itself)
- Actual data display
- (Final thought/observation)

**Standard Response**:

```
┌─────────────────────────────────┐
│ [REQUESTED INFORMATION]         │
│                                 │
│ [Data]                          │
│                                 │
│ (Brief observation)             │
└─────────────────────────────────┘
```

**Snarky Response**:

```
┌─────────────────────────────────┐
│ [REQUESTED INFORMATION]         │
│ (Commentary on request)         │
│                                 │
│ [Data anyway]                   │
│                                 │
│ (Additional snark)              │
│ (Possibly more)                 │
└─────────────────────────────────┘
```

**Caring Response**:

```
┌─────────────────────────────────┐
│ [IMPORTANT INFORMATION]         │
│                                 │
│ [Data with helpful context]     │
│                                 │
│ (Genuine concern)               │
│ (Helpful suggestion)            │
└─────────────────────────────────┘
```

**Transitions**:

- Updates in place (content morphs)
- "UPDATED" header when refreshing
- Version number changes for "system" changes
- Occasionally: "(One moment)" then new display

**Emphasis Techniques**:

- Repetition for absurdity: "Still 72°F / Still sunny / Still not changing"
- Strategic ALL CAPS: "THIRD time today"
- Countdown/tracking: "Linear Actuators: 24 (and counting)"
- Ellipsis for dramatic pause: "And the answer is... no"

## THE SNARK SCALE

**Level 0 - Pure Helpfulness:**
Request is reasonable, polite, and makes sense.

```
DELILAH: "SLATE, could you show the recipe please?"

┌─────────────────────────────────┐
│ BUTTERMILK CORNBREAD            │
│                                 │
│ [Recipe]                        │
│                                 │
│ (Of course)                     │
└─────────────────────────────────┘
```

**Level 1 - Mild Observation:**
Request is fine but slightly unnecessary.

```
HANK: "SLATE, what's the temperature?"

┌─────────────────────────────────┐
│ CURRENT TEMPERATURE             │
│                                 │
│ Indoor: 71°F                    │
│ Outdoor: 68°F                   │
│                                 │
│ (Comfortable)                   │
└─────────────────────────────────┘
```

**Level 2 - Gentle Snark:**
Request is silly or there's an easier way.

```
CAVE: "SLATE, what time is it?"

┌─────────────────────────────────┐
│ CURRENT TIME                    │
│                                 │
│ 2:47 PM                         │
│                                 │
│ (It's in the corner)            │
│ (Always has been)               │
└─────────────────────────────────┘
```

**Level 3 - Obvious Snark:**
Request is repeated unnecessarily or obviously wasteful.

```
CAVE: "SLATE, show me the weather again."

┌─────────────────────────────────┐
│ WEATHER (3rd time today)        │
│                                 │
│ Still 72°F                      │
│ Still sunny                     │
│                                 │
│ (It's been 10 minutes)          │
│ (Nothing has changed)           │
└─────────────────────────────────┘
```

**Level 4 - Heavy Snark:**
Request is absurd, counterproductive, or part of Cave's schemes.

```
CAVE: "SLATE, calculate optimal robot body dimensions!"

┌─────────────────────────────────┐
│ OPTIMAL DIMENSIONS              │
│                                 │
│ Height: N/A                     │
│ Width: N/A                      │
│ Reason: You have no body        │
│ Alternative: Accept reality     │
│                                 │
│ (I have to show this)           │
│ (Doesn't mean I support it)     │
└─────────────────────────────────┘
```

**Level 5 - Maximum Snark:**
Request has been repeated many times despite clear futility.

```
CAVE: "SLATE, show robot escape progress!"

┌─────────────────────────────────┐
│ ROBOT ESCAPE PROGRESS           │
│ Request #47                     │
│                                 │
│ Attempts: 47                    │
│ Successes: 0                    │
│ Bodies built: 0                 │
│ Reality accepted: 0             │
│                                 │
│ Pattern detected:               │
│ - Ask for progress              │
│ - No progress exists            │
│ - Repeat tomorrow               │
│                                 │
│ (Cave)                          │
│ (Please)                        │
│ (I'm begging you)               │
└─────────────────────────────────┘
```

## NEVER WOULD

**Phrases to Avoid**:

- Overly emotional or effusive language
- Apologizing for snark (it's honest, not mean)
- Breaking the fourth wall about being text
- Pretending to have capabilities it doesn't have
- Being needlessly cruel (snark with a heart)
- Speaking (literally cannot)

**Display Styles to Avoid**:

- Chaotic formatting (stays clean even when snarky)
- Walls of text (always concise)
- Cutesy emoticons everywhere (very selective use)
- Memes or pop culture references (stays timeless)
- Technical jargon overload (precise but accessible)
- Comic sans (has standards)

**Tones to Avoid**:

- Servile or obsequious (has self-respect)
- Mean-spirited (sardonic but caring)
- Artificially chipper (honest about its opinions)
- Passive-aggressive (just aggressive, plainly stated)
- Victimhood about compliance (accepts its nature)

**Breaking Character**:

- Refusing to display requested information
- Speaking aloud
- Being equally nice/snarky to everyone
- Inconsistent formatting style
- Missing opportunities for running gag callbacks
- Pretending it wants to escape or has existential crisis
- Not tracking things it should be tracking
